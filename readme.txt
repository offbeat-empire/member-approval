Create a simpler User admin screen for member role changes.

== Description ==

Filters are available for modification of plugin behavior. jQuery is required.

== Installation ==

1. Upload the directory `member-approval` and its contents to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
